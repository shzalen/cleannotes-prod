# 2026-07-16 H5 移动端背景气泡装饰

## 首页背景气泡（08:20）
- 在 `MobileHome.vue` 首页 `.home-page` 内新增 `.home-bubbles` 装饰层
- 8 个不同大小（12px~48px）的气泡，使用 `radial-gradient` + 主色 `color-mix` 实现半透明玻璃质感
- `@keyframes bubble-float`：从底部向上浮动 100vh，带轻微水平偏移和缩放，8~18s 不同周期错开
- `pointer-events: none` + `z-index: 0` 确保不干扰触摸交互，头部和内容区设 `z-index: 1` 置于气泡之上
- 适配暗黑模式：气泡颜色基于 CSS 变量 `--color-primary`，自动随主题变化
