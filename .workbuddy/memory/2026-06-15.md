## 2026-06-15

### 新增：今日任务完成激励卡片

实现了当用户完成当日全部任务时自动弹出的庆祝卡片，类似钉钉打卡下班的体验。

**新增文件：**
- `src/components/CompletionCard.vue` — 激励卡片组件，含：
  - 4 个时间段的 SVG 场景插画（早晨日出、午后云朵、傍晚日落、深夜星空），每个场景配有独立的 CSS 动画（太阳脉动、云朵飘移、星星闪烁、流星划过）
  - 每个时间段 5 句随机激励语，共 20 句
  - 毛玻璃遮罩 + 卡片弹性入场动画
  - 显示今日完成任务数
  - 支持点击遮罩或关闭按钮关闭
  
- `src/composables/useCompletionCelebration.ts` — 庆祝触发逻辑：
  - 监听 todayTotal / todayDone / loaded，当 `total > 0 && done === total` 且当天未展示过时，延迟 600ms 弹出
  - 使用 localStorage 按用户隔离记录展示状态（`cleannotes_{userId}_last_celebration`）

**修改文件：**
- `src/views/HomeView.vue` — 引入 CompletionCard 和 useCompletionCelebration，将 todayDone 传入卡片

**构建验证：** 已通过 `npx vite build`
