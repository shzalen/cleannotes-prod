### PyInstaller 打包替代方案 (16:04)
- **核心问题**：Tauri 在 Rust 编译时将前端嵌入 exe，cargo.exe 被杀毒拦截后无法更新 exe 中的前端
- **解决方案**：Python pywebview + PyInstaller，完全绕开 cargo.exe
  - launcher.py：使用 pywebview 创建原生 WebView2 窗口，bottle 提供 HTTP 静态文件服务
  - 加载 Vue dist/ 目录的最新前端构建产物
  - PyInstaller --onefile --windowed 打包为独立 exe
- 安装依赖：`pip install pywebview PyInstaller`
- 产物：dist-app/清记.exe (~18MB) + dist-app/清记_0.1.0_x64-setup.exe (~18MB)

### PyInstaller 也被杀毒拦截 → 免 exe 方案 (16:12)
- PyInstaller 生成的 exe 也被杀毒软件拦截，彻底放弃生成 exe
- **最终方案**：NSIS 安装 web 静态文件 + Python 启动脚本，零可执行文件
  - launcher.py：pywebview 原生窗口，自动查找 web/ 或 dist/ 目录
  - 清记.vbs：VBScript 启动器，调用 pythonw.exe 无控制台窗口
  - 清记.bat：备用 bat 启动器（带控制台回退）
  - NSIS 安装到 `%LOCALAPPDATA%\CleanNotes\`，桌面快捷方式指向 wscript.exe + VBS
- 安装程序从 18MB 降至 304KB（不再包含 Python 运行时）
- **前提**：目标机器需安装 Python 3.10+ 并 pip install pywebview
