# 2026-07-08 工作记录

## 登录页增加密码登录（加密存储）
- 需求：登录页增加「密码登录」，密码存储须加密
- 实现要点：
  - 新建 `src/utils/crypto.ts`：基于浏览器原生 Web Crypto 的 PBKDF2-HMAC-SHA256（随机盐 16B + 21 万次迭代，verifyPassword 用恒定时间比较），零依赖
  - `cleannote_users` 表新增 `password_hash` / `password_salt`（可空）；`supabase-schema.sql` 已含，`migration-password.sql` 供已有库 ALTER TABLE 加列
  - `src/services/auth.ts`：新增 `findUserByPhoneWithCredentials`（返回 hash/salt）、`setPassword(userId, hash, salt)`；`registerUser` 支持写入密码；`rowToUser` 不泄露密码字段
  - `src/stores/auth.ts`：新增 `submitPassword(phone, password, confirm?)`，统一处理「新用户注册设密 / 遗留账号首次设密 / 已设密校验」三路径；明文密码仅在本地校验，绝不传输或落库
  - `src/views/LoginView.vue`：增加「密码登录 / 验证码登录」双 Tab，密码表单含显隐切换 + 确认密码（首次设置时显示）
- 顺带修复 2 处既有类型错误（`ReportPoster.vue` clonedEls 类型、`weeklyReport.ts` generateReportContent 参数），它们阻断了整个 vue-tsc 构建
- 构建注意：本沙箱内 `vite build` 的 emptyDir 会触发 WorkBuddy 批量删除保护（dist/assets >50 文件），需先 `rm -rf dist` 再构建；用户本机无此拦截，正常 `npx vite build` 即可
