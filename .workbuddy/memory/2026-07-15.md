## 触控防误触优化 (useTouchInteraction)
- 问题：首页下拉时容易误触任务项（touch 位移极小被判定为 tap 打开详情）
- 方案一：位移阈值拦截（> 15px 视为滚动/下拉，取消一切交互）
- 新增 `src/mobile/composables/useTouchInteraction.ts`
  - 判定规则：位移 > 15px → 取消；位移 ≤ 15px + duration < 1s → tap；位移 ≤ 15px + duration ≥ 1s → long-press（带振动）
  - 泛型设计 `useTouchInteraction<T>`，返回 `{ handleTouchStart, handleTouchMove, handleTouchEnd }`
- MobileHome.vue 和 MobileCalendar.vue 任务项触控已改为使用 composable
- 构建成功：2862 modules, 28s；robocopy 同步 132 文件到生产目录
