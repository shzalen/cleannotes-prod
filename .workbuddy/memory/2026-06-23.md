# 2026-06-23 工作日志

## 任务弹窗增加实际执行时间补录功能
- **需求**：用户线下已开始/完成任务但忘记点击小圆点，需在弹窗中手动补录时间
- **Store 层** (`src/stores/task.ts`)：
  - `updateTask` patch 类型扩展了 `inProgressAt` 和 `completedAt`
  - 自动时间戳逻辑：仅 patch 中未显式传入该字段时才自动填充（`!('inProgressAt' in patch)` / `!('completedAt' in patch)`）
  - 小圆点调用路径不受影响（不传时间字段，行为不变）
- **弹窗层** (`src/components/TaskEditModal.vue`)：
  - 新增 `inProgressAtInput` / `completedAtInput` ref（datetime-local 格式）
  - 新增 `toDatetimeLocal` / `fromDatetimeLocal` 互转函数
  - `showTimestamps` computed：仅编辑非只读时显示
  - `openEdit` 时回填已有值
  - `doSave` 时仅非空传入 patch
  - UI：新增"实际执行时间"区域（浅灰底色卡片），含两个 datetime-local 输入框 + 提示文案
  - 待办状态下 completedAtInput 禁用
  - 位置：状态行→实际时间→开始日期/时间节点
- 构建通过，零错误
