## 周报海报导出修复 + 布局重构
- **海报导出修复**：html2canvas-pro 无法解析 `var(--color-*)` 和 `color-mix()`，导致导出 PNG 背景透明、内容缺失
  - 修复方案：`onclone` 回调中预收集所有元素的 `getComputedStyle` 值，内联到克隆 DOM；同时在克隆文档注入 `:root` CSS 变量解析值
- 海报宽度 440px → 480px → 540px（增加开始时间+耗时列后需更多空间）
- 涉及属性：backgroundColor/color/borderColor/backgroundImage/boxShadow/opacity/fill/stroke 等
