## XMind 风格思维导图颜色升级

- 替换 markmap 默认扁平配色为 XMind 风格分支族谱配色方案：
  - **8 色分支调色板**：Coral/Blue/Green/Amber/Purple/Teal/Rose/Cyan（#E74C3C 等）
  - **根节点**：#34495E（深蓝灰），白色粗体文字
  - **color 回调 `nodeColorFn(node)`**：解析 node.state.path 提取分支索引（如 "0.2.1" → 分支 2），depth=1 用原色，depth≥2 用 `deriveChildColor()` 递增减饱和/增明度
  - **style 回调 `mindmapStyleFn(id)`**：注入作用域 CSS — 圆角 rect 节点（rx:6-8px）、drop-shadow 阴影、2.5px 链接线圆头收尾、分层字体大小（depth0:16px/700, depth1:15px/600, depth2:14px, depth3+:13px）
  - **hover 增强**：节点阴影加深，链接颜色平滑过渡
- `deriveChildColor(hex, depth)`：hex→HSL→减饱和 8%/级 + 增明度 7%/级，保证深层次子节点柔和但仍与父分支同色系
- `hexToHsl(hex)`：3/6 位 hex → [H°, S%, L%]
- 应用于 `renderInline()` 和 `renderPreview()` 两处 `Markmap.create()` 调用
